자바 프로그래밍 포트폴리오 용
도서 대여 관리 프로그램입니다.

도서 목록, 대여 현황, 회원 관리, 책 바구니 페이지로 구성되어 있습니다.

도서 목록과 회원 관리 페이지에선 도서와 회원의 정보를 추가, 수정, 삭제할 수 있습니다.

도서 목록에서 데이터 테이블 안의 테이블을 클릭하고 담기 버튼을 누르면 책 바구니 목록으로 도서가 담아집니다.

책 바구니 페이지에서 도서를 클릭하고 대여 버튼을 누르면 도서의 상태가 대여로 바뀌며 대여 현황에 기록됩니다.

대여 현황에서 반납 처리를 할 수 있습니다.

원하는 컬럼명에 맞춰서 검색이 가능합니다.

자바 1.8 버전이며
DB는 오라클 SQL을 사용했습니다.

폴더 내 첨부된 .sql 파일을 열어 시퀀스와 테이블을 만들고, DBConnect.class의 jdbc url,id,passwd를 바꾸면 DB연결을 할 수 있습니다.
